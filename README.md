# miaulacucei

El sistema contribuye a los objetivos organizacionales ya que el programa pide
específicamente cuales aulas están vacías, la tecnología actual tiene las
herramientas necesarias para el desarrollo así mismo entra en el presupuesto
aceptable ya que es adaptable a otros sistemas ya sea para complementarlo
o para integrando (ejemplo siiau)
========================================
Implementación del estudio de viabilidad
si el sistema no es implementado podría perderse tiempo si se revisa
físicamente salón por salón ya que no hay una herramienta (software) que nos
ayude a encontrar uno de manera rápida y la capacidad del salón.
=========================================

El sistema propuesto nos permite ahorra tiempo a organizar el tiempo y
optimizar el sistema (mejorar la calidad de clase).
Podríamos enfrentarnos a un usuario que no se conforme con las opciones que
le otorga el software. Sin embargo, con las herramientas actuales es posible el
desarrollo y optimizarlo al igual para agilizar la asignación de aulas
dependiendo de los usuarios.
========================================
Conclusiones:
ES viable porque se pueden aprovechar las herramientas actuales y se puede
complementar el sistema actual.
